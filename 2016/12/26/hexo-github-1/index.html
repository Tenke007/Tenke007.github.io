hexo+github教程之一：搭建hexo博客

发表于 2016-12-26 | 分类于 hexo | |   阅读数

前言

本人几年前就想搭建一个属于自己的博客，刚好前段时间看见很多大神都用hexo搭建自己的个人博客，于是乎自己也动手折腾了一番，搞了两天终于弄出一个还过得去的博客。废话不多说，下面开始动手！

第一步

1、git下载与安装

git的下载安装应该没什么难度，不知道git怎么下载的请自行度娘，反正安装的时候一直默认配置就行了，一直next。安装好后，鼠标右键任意地方应该会出现如图，就说明已经安装成功了。

2、安装node.js

node.js的官网好像被墙了，反正我是费了好大劲才下载好。node.js地址，选择最新版本。然后也是一路next就行了。

1、鼠标右键任意地方，选择Git Bash ，使用以下命令安装hexo(ps：以下命令中的$符号只为了让教程和实际看起来一致，实际输入命令只需输入$后面的命令即可)

执行完命令应该会有C:\Users\Administrator\AppData\Roaming\npm这个目录。

自己可以发一篇新的文章
